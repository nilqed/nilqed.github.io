Patternable(R)

An object S is Patternable over an object R if S can lift the conversions from R into Pattern(Integer) and Pattern(Float) to itself.

convert : % -> Pattern(Float) if R has ConvertibleTo(Pattern(Float))
from ConvertibleTo(Pattern(Float))
convert : % -> Pattern(Integer) if R has ConvertibleTo(Pattern(Integer))
from ConvertibleTo(Pattern(Integer))

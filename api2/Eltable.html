An eltable over domains D and I is a structure which can be viewed as a function from D to I. Examples of eltable structures range from data structures, e.g. those of type List, to algebraic structures, e.g. Polynomial.

elt : (%, D) -> I

elt(u, i) (also written: u.i) returns the element of u indexed by i. Error: if i is not an index of u.

HermiteIntegrate : (R, Mapping(UP, UP), R) -> Record(answer : R, logpart : R)

HermiteIntegrate(f, ', d0) returns [g, h] such that f = g' + d0*g + h and h has a only simple finite normal poles. Note: d0 must be integral.

This category provides operations on ranges, or segments as they are called.

(l..h)+i returns segment l+i .. h+i.

+ : (S, %) -> % if S has AbelianSemiGroup

i+(l..h) returns segment i+l .. i+h.

(l..h)-i returns segment l-i .. h-i.

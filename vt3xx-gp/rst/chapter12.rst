.. _S12:

12 Combining ReGIS Commands
===========================

.. _S12.1:

Drawing Complex Images
----------------------

Figures `12-1 <#F12-1>`__ through `12-3 <#F12-3>`__ show examples of
complex graphic images you can build with ReGIS commands. Together,
these three figures show most of the ReGIS features available.

`Figure 12-1 <#F12-1>`__ uses screen, write control, and text commands
to define basic command values. `Figure 12-2 <#F12-2>`__ uses the values
in effect at the end of `Figure 12-1 <#F12-1>`__. `Figure
12-3 <#F12-3>`__ uses the values in effect at the end of `Figure
12-2 <#F12-2>`__.

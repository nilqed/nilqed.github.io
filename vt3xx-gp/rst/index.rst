.. _BOOKTITLE:

VT330/VT340 Programmer Reference Manual
Volume 2: Graphics Programming
=======================================

.. container::
   :name: COVER

   |To the Contents Page|

Download this entire book as a `gzipped tar file <../vt3xx-gp.tgz>`__
(528 KB) or `zip file <../vt3xx-gp.zip>`__ (567 KB).

This document is a reproduction of the second edition of this book,
published by Digital in May 1988. The first edition was published in
March 1987. The largest changes in this edition are:

-  There are no longer any default reports when locator buttons are
   released.
-  When two sessions are displayed at the same time, using windows, the
   locator cursor can no longer roam freely over the whole screen.
-  There is a new programming example for sending sixel graphics to the
   LN03 printer.
-  There is a new appendix, detailing the changes between this terminal
   and previous Digital graphics terminals.

The first edition of this manual was contributed by Steve Harrison.
